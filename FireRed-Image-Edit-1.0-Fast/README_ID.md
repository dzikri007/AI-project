# Owi Edit - Image Editing dengan AI

Owi Edit adalah aplikasi editing gambar canggih yang menggunakan AI untuk mentransformasi gambar sesuai instruksi Anda.

## Fitur Utama 🎯

- ✨ **Editing Berbasis AI** - Ubah gambar dengan instruksi natural language
- 🖼️ **Multi-Image Support** - Gunakan beberapa gambar sebagai referensi
- 🎨 **Prompt Cepat** - Kumpulan prompt siap pakai untuk inspirasi
- ⚙️ **Pengaturan Lanjutan** - Kontrol seed, guidance, dan steps
- 🌐 **Interface Responsif** - Bekerja baik di desktop dan mobile
- 📖 **Panduan Interaktif** - Tutorial lengkap dalam aplikasi

## Bagaimana Cara Menggunakan? 🚀

1. **Unggah Gambar**: Klik atau seret gambar ke area upload
2. **Tulis Instruksi**: Ketik apa yang ingin Anda ubah
3. **Jalankan**: Klik tombol "Edit Gambar"
4. **Unduh**: Simpan hasil yang Anda sukai

Contoh instruksi:
- "Ubah menjadi gaya anime"
- "Tingkatkan kualitas dan buat lebih tajam"
- "Tambahkan efek cinematic dengan cahaya hangat"
- "Ubah menjadi lukisan minyak"

## Teknologi 🔧

- **Model**: FireRed-Image-Edit-1.1 (Qwen-Image-Edit Rapid)
- **Framework**: Gradio + Python
- **Optimasi**: 4-Step Fast Inference
- **Hardware**: GPU-accelerated (CUDA)
